# Prasenjit_SciArt-2nd-Assignment-
Protocol: Creating an Aarskog-Scott Syndrome Infographic in BioRender
This document provides a detailed, step-by-step protocol for recreating the medical infographic for Aarskog-Scott Syndrome, originally created using BioRender. The goal is to produce a comprehensive visualization combining genetic inheritance, molecular biology (DNA mutation), and clinical symptoms.

1. Preparation and Setup
Open BioRender: Log into your BioRender account and start a new, blank canvas.

Establish Layout Strategy: Plan for a single-column, vertical layout with distinct, sequential sections separated by headers. Use a neutral background (white).

Title:

Select the Text tool (T).

Place a bold, main title at the top-left: Aarskog-Scott Syndrome. Include the characteristic vertical bar (|) to the left of the title as a design element.

2. Section: Inheritance Pattern (Top-Main)
Add Descriptive Text Box: Add a shaded grey text box directly beneath the title. Inside, enter the descriptive text: "Aarskog-Scott Syndrome is caused by mutations in the gene FDG1, located on the short arm of the X chromosome (Xp11.2)."

Add Male and Female Figures:

Search the Icon library for: "man outline" (for the Father) and "woman outline" (for the Mother carrier).

Place them side-by-side below the grey text box. Label them "Father" and "Mother (carrier)".

Search for "boy" or "son" to add a smaller child figure on the right.

Add Sex Chromosome Icons:

Search for "chromosome" and select the distinct paired X and XY icons.

Father (X Y): Group the "X" and "Y" icons. Color the X chromosome outline/fill with white and dark grey stripes. Color the Y chromosome smaller and solid white.

Mother (X X): Search for "X chromosome pair". This icon should clearly show two X's. Leave one "healthy" X as white/striped and color the other "carrier" X (the one carrying the mutation) with red/striped fill to denote the affected gene.

Connect with Logic:

Place a large X symbol (using standard text or a line icon) between the father and mother icons to indicate a cross.

Draw an arrow (→) leading from the mother figure/chromosomes to the affected son.

Son's Chromosomes:

Repeat the chromosomes for the son (X Y), using the same visual coding. Crucially, the son's X chromosome (the one he inherited from the carrier mother) should be the red/striped one, and the Y from the father is the small white one.

3. Section: DNA Mutation Details (Middle)
Add Header Label: Add a descriptive header in a light pink shaded box below the inheritance section: "Mutations in the FDG1 gene mainly missense mutation (type of point mutation)".

Depict Normal DNA Strand:

Search for "DNA strand icon" or "base pairs" and select a clean, multi-colored base pair block.

Use BioRender’s labeling system to manually create nucleotide letters (A, T, G, T, G, A, C, T) above the strand. Assign consistent colors for each base pair (e.g., A=purple, T=orange, G=blue, C=darker blue/purple) using the color picker tool.

Label this element: Normal DNA strand.

Depict Mutated DNA Strand:

Duplicate the entire normal DNA strand object.

Crucial Step: Manually edit the replicated strand at the specific point of mutation. Change the third base G (blue) in the sequence to C (a different blue or red, indicating the mutation). The rest of the sequence (A T ... G A C T) must remain identical.

Label this element: DNA strand with Point mutation.

Connect with Logic: Draw an arrow (→) showing the transition from the normal strand to the mutated strand.

Add Functional Note: Add a line of plain text below the DNA strands: The gene FDG1 codes for Rho/Rac guanine exchange factor.

4. Section: Clinical Symptoms (Bottom)
Add Header Label: Add another light pink shaded header box: Symptoms.

Create Symptoms Container:

Use the Rectangle Tool (R) to draw a large, rounded rectangle with a thin border and light grey fill. Place this on the right side of the canvas.

Create a vertical list of descriptive text fields on the left side of the new container.

Populate the Symptoms List (Right Container):

Search for "eye" and select a human eye icon. Duplicate it to create a pair of eyes.

Draw a horizontal line with arrows on both ends (←--→) between the eyes and label it: Ocular hypertelorism.

Search for "droopy eyelid" or "ptosis" icon. Add this below the eyes and label it: Ptosis.

Search for "man profile" or "male head outline." Place this on the far right. Use a line/pointer to highlight the low-set hairline on the forehead and label it: Widow’s peak.

Add Icons to the Left List:

Search for "person icon," "human outline," and a "fetal outline" (or "embryo").

Stack these three icons vertically to the left of the rounded symptoms container.

Add corresponding text labels for each icon: Facial abnormalities, Skeletal differences, and Developmental issues.

Draw a single horizontal connector line leading from the "Facial abnormalities" text label to the entire symptoms container on the right.

5. Final Polishing
Check Alignment: Group related elements and use the BioRender alignment tools (Distribute Vertically, Align Centers) to ensure a professional, clean look.

Standardize Color and Line Weights: Verify that all lines, text boxes, and icons use consistent line weights and color palettes to enhance legibility.
